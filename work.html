Meet Me: Web App

This is a serverless, progessive web application (PWA) with React using a test-driven development (TDD) technique. The application uses Google Calendar API to fecth upcoming events. Some of the projects' features are being able to filter events by city, specify the number of events searched, use the app when offline, add an app shortcut to your home screen, and displays charts visualizing event details.

    Technologies:

  • React
  • AWS Lambda
  • Jest-Cucumber
  • Puppeteer
  • Atatus
  • Recharts
  • Google APIs
  • Google OAuth 2.0
